Bienvenido al Proyecto

Este proyecto implementa un pipeline ETL completo que procesa datos financieros (gastos y presupuestos) para calcular el KPI de ejecución presupuestaria por área organizacional. El sistema permite identificar áreas en riesgo de sobrepasar su presupuesto y optimizar la gestión financiera mediante un análisis detallado y automatizado.

🏗️ Arquitectura del Proyecto

El pipeline ETL implementa una arquitectura de 3 capas (Medallion Architecture) que garantiza la trazabilidad, calidad y escalabilidad de los datos:

🔵 Capa Bronce (Raw)

  • Datos crudos sin transformar
  • Trazabilidad completa (batch_id, event_id)
  • Formato: Apache Parquet
  • Inmutable y auditable
  • Registro de origen (_source_file)

⚪ Capa Plata (Clean)

  • Datos validados y normalizados
  • Sin duplicados (clave natural)
  • Tipos de datos correctos
  • Sistema de cuarentena
  • Reglas de negocio aplicadas

🟡 Capa Oro (Gold)

  • KPIs calculados
  • Métricas de negocio
  • Agregaciones optimizadas
  • Vista analítica en SQLite
  • Listo para consumo

📊 KPI Principal: Ejecución Presupuestaria

Fórmula del KPI

KPI_Ejecución = (Gasto Acumulado / Presupuesto Anual) × 100

Interpretación del KPI: Este indicador permite identificar el porcentaje de presupuesto ejecutado por cada área organizacional, facilitando la toma de decisiones preventivas y la optimización de recursos.

< 70%

Bajo consumo

70-89%

Normal

90-100%

En riesgo

> 100%

Crítico

✨ Características Principales

🔄 Idempotencia

Reprocesar los mismos datos no genera duplicados gracias al sistema de batch_id único por ejecución.

🔍 Trazabilidad

Cada registro mantiene su origen completo (_source_file, _batch_id, _event_id) para auditoría total.

⚠️ Sistema de Cuarentena

Los registros inválidos se guardan con causa documentada para análisis posterior.

🎯 Deduplicación

Política "último gana" basada en clave natural (fecha, área, partida).

💰 Precisión Financiera

Uso de DECIMAL(18,2) para evitar errores de redondeo en importes monetarios.

📊 Reportes Automáticos

Generación automática de reportes en Markdown y HTML con cada ejecución.
