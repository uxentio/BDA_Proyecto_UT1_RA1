Bienvenido al Proyecto

Este proyecto implementa un pipeline ETL completo que procesa datos financieros (gastos y presupuestos) para calcular el KPI de ejecución presupuestaria por área organizacional.

🏗️ Arquitectura del Proyecto

Pipeline ETL de 3 capas:

🔵 Bronce (Raw)
Datos crudos con trazabilidad completa
⚪ Plata (Clean)
Datos validados y normalizados
🟡 Oro (Gold)
KPIs y métricas analíticas

🛠️ Tecnologías Utilizadas

Python 3.11+
Lenguaje principal del pipeline
Pandas & NumPy
Procesamiento de datos
Apache Parquet
Almacenamiento columnar eficiente
SQLite
Base de datos analítica

📊 KPI Principal

KPI_Ejecución = (Gasto Acumulado / Presupuesto Anual) × 100

Este KPI permite identificar áreas en riesgo de sobrepasar el presupuesto asignado y optimizar la gestión financiera.
